How It Works

Simple, effective, automatic.

1

Install & Grant Permission

Download CatPaws and grant Input Monitoring permission in System Settings. This allows the app to detect keyboard patterns.

2

Cat Approaches Keyboard

When your cat steps on the keyboard, CatPaws detects the characteristic pattern of multiple adjacent keys being pressed simultaneously.

3

Keyboard Locks Instantly

CatPaws blocks all keyboard input and shows a notification. Your work is safe from random cat-generated text.

4

Cat Leaves, You Continue

When the keys are released, CatPaws automatically unlocks. Resume typing as if nothing happened.
